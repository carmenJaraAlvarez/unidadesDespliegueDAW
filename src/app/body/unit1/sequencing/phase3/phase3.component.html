Fase de Síntesis o Cierre: Compartir, evaluar y reflexionar sobre nuestros logros

  1. Presentación de la landing page y reflexión final sobre el trabajo colaborativo
    Descripción: Los grupos presentan su landing page y reflexionan sobre el proceso colaborativo.
    Recursos: Ordenadores, conexión a Internet, pantalla digital, presentación en PowerPoint.
    Agrupamiento: Presentación grupal.
  2. Rúbrica de evaluación
    Criterios:
    • Calidad del código y la funcionalidad: El código está perfectamente estructurado, limpio y bien comentado. La landing page es completamente funcional.
    • Trabajo en equipo y colaboración: El equipo ha trabajado de manera excelente, con una distribución de tareas equilibrada y buena comunicación.
    • Uso de Git y GitHub: Se ha utilizado Git y GitHub de manera excelente, con commits regulares, ramas bien gestionadas y pull requests claros.
    • Implementación de DevOps (CI/CD y despliegue continuo): El proceso de CI/CD se ha implementado correctamente con GitHub Actions y la landing page se ha desplegado sin errores.
    La evaluación será colectiva, y tendrá en cuenta tanto la calidad técnica del producto final como el grado de colaboración en el equipo con ayuda de la rúbrica.
